## Get Your API Key

To access the Cleanview API, you'll need an API key. During the beta phase, API keys are manually provisioned.

<Steps>
  <Step title="Request Access">
    Contact our team to request an API key by emailing [api@cleanview.energy](mailto:api@cleanview.energy?subject=API%20Key%20Request) with:
    - Your name and company
    - Intended use case for the API
    - Expected request volume
  </Step>
  <Step title="Receive Your Key">
    You'll receive your API key in the format: `cv_pub_[32 random characters]`
    
    Example: `cv_pub_KJ3x9mN2pQ8rS7tV5wY6zA1bC4dE0fG2`
  </Step>
  <Step title="Store Securely">
    Keep your API key secure and never expose it in client-side code or public repositories.
  </Step>
</Steps>

## Make Your First Request

Once you have your API key, you can start making requests to the Cleanview API.

### Authentication

Include your API key in the `Authorization` header of all requests:

```bash
curl -H "Authorization: Bearer cv_pub_your_api_key_here" \
  https://api.cleanview.energy/api/v0/eia-860m
```

### Basic Request Examples

<AccordionGroup>
  <Accordion icon="database" title="Get EIA 860M Data">
    ```bash
    curl -H "Authorization: Bearer cv_pub_your_api_key_here" \
      "https://api.cleanview.energy/api/v0/eia-860m?limit=10"
    ```
    
    ```javascript
    const response = await fetch('https://api.cleanview.energy/api/v0/eia-860m?limit=10', {
      headers: {
        'Authorization': 'Bearer cv_pub_your_api_key_here'
      }
    });
    const data = await response.json();
    ```
  </Accordion>
  
  <Accordion icon="network-wired" title="Get Interconnection Queue Data">
    ```bash
    curl -H "Authorization: Bearer cv_pub_your_api_key_here" \
      "https://api.cleanview.energy/api/v0/interconnection/ercot?limit=5"
    ```
    
    ```python
    import requests
    
    headers = {'Authorization': 'Bearer cv_pub_your_api_key_here'}
    response = requests.get(
        'https://api.cleanview.energy/api/v0/interconnection/ercot?limit=5',
        headers=headers
    )
    data = response.json()
    ```
  </Accordion>
</AccordionGroup>

### Response Format

All API responses follow a consistent format:

```json
{
  "success": true,
  "data": [...],
  "metadata": {
    "total": 1000,
    "limit": 10,
    "offset": 0
  },
  "pagination": {
    "next": "https://api.cleanview.energy/api/v0/eia-860m?limit=10&offset=10",
    "previous": null
  }
}
```
