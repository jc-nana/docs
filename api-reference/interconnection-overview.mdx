# Interconnection Queues API

Access real-time interconnection queue data from all 7 major ISOs/RTOs in the United States. Our API standardizes data from different grid operators into consistent formats while preserving ISO-specific details.

## Quick Start

```bash
# Get solar projects from CAISO
curl -H "x-api-key: your_api_key" \
  "https://api.cleanview.co/api/v0/interconnection/caiso?technology=Solar&limit=50"

# Get wind projects from ERCOT  
curl -H "x-api-key: your_api_key" \
  "https://api.cleanview.co/api/v0/interconnection/ercot?technology=Wind&limit=50"
```

Use consistent parameters across all endpoints:

```bash

GET /api/v0/interconnection/{iso}?limit=50

# Filter by technology
GET /api/v0/interconnection/{iso}?technology=Solar&limit=50

# Filter by capacity and location  
GET /api/v0/interconnection/{iso}?minCapacity=100&state=CA&limit=50

# Multiple filters
GET /api/v0/interconnection/{iso}?technologies=Solar,Wind&status=Planned&limit=50
```
